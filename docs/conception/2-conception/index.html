Les boucles

On appelle boucle le questionnement répétitif d’un ensemble de questions. Une boucle ne peut concerner que des séquences (ou sous-séquences) entières. On appelle cet ensemble de questions une "occurrence" (de la boucle). L'atelier de conception propose deux types de boucles :

  • une boucle dite simple pour laquelle on définit un nombre minimum et un nombre maximum d’occurrences affichables, en absolu (numérique) ou avec une variable collectée ou calculée ;
  • une boucle dite liée qui se base sur les occurrences d'une boucle précédente.
